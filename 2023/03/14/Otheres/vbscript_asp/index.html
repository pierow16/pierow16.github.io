vbscript语法摘抄

  2. 可以通过 Dim、Public 或 Private 语句声明 VBScript 变量

条件语句

  1. If 语句 - 假如您希望在条件为 true 时执行一系列的代码，可以使用这个语句 .
    多个操作时候End If必须 不然报错
  2. If…Then…Else 语句 - 假如您希望执行两套代码其中之一，可以使用这个语句
  3. If…Then…ElseIf…Then..Else..End If 语句 - 假如您希望选择多套代码之一来执行，可以使用这个语句
  4. Select Case 语句 - 假如您希望选择多套代码之一来执行，可以使用这个语句

    循环

  5. For…Next 语句 - 运行一段代码指定的次数.

    可以通过step关键词规定计数变量递增或递减的步进值
    通过 Exit For 关键词 相当于return

  6. For Each item IN obj ..Next 语句 - 针对集合中的每个项目或者数组中的每个元素来运行某段代码
  7. Do…Loop 语句 - 运行循环，当条件为 true 或者直到条件为 true 时
  8. While…Wend 语句 - 不要使用这个语句 - 请使用 Do…Loop 语句代替它
